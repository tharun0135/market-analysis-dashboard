# Algorithmic Trading Strategy Backtester

Python backtesting engine and interactive dashboard for evaluating algorithmic trading strategies. This project serves as the core infrastructure for a Fintech market analysis tool, utilizing FastAPI for the backend and a lightweight, dynamic frontend.

Tech Stack

Backend: Python 3.11+, FastAPI, Pandas, NumPy, yfinance, Pytest
Frontend: HTML5, Vanilla JavaScript, TailwindCSS, Chart.js
Server: Uvicorn
